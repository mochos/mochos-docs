## 🎮 ¿Cómo jugar?

Para entrar en Tulacraft debes haber sido invitado por uno de los creadores y seguir las instrucciones a continuación según si eres premium o no. Solicita a la persona que te invitó que te de acceso.

<Tabs>
  <Tab title="PolyMC (Fácil)">
    <Warning>
      Es indispensable que tengas Java instalado. Versión recomendada: 21. Descárgalo de [este enlace](https://www.oracle.com/java/technologies/downloads/#java21).
    </Warning>
    <Steps>
      <Step title="Instala PolyMC">
        [Descarga desde la web oficial](https://polymc.org/download/)
      </Step>
      <Step title="Si es primera vez que abres el Launcher debes configurar el Java y agregar tu nombre de usuario">
        <AccordionGroup>
          <Accordion title="Configurar Java">
            En la primera ejecución te preguntará sobre el ejecutable de Java. Debes seleccionar la versión recomendada y asignarle suficiente memoria:

            ![Configurar Java Web](/tulacraft/imagenes/configurar-java.webp)
          </Accordion>
          <Accordion title="Establecer nombre de usuario">
            En la ventana principal del launcher en la parte superior derecha dale clic a "Perfiles" y luego a "Gestionar cuentas...".

            <img
              src="/tulacraft/imagenes/perfiles.webp"
              alt="Perfiles Web"
              title="Perfiles Web"
              style={{ width:"61%" }}
            />

            Luego dale clic a "Agregar cuenta sin conexión" y en la ventana que se abre agrega tu nombre de usuario con el que entrarás al servidor.

            <img
              src="/tulacraft/imagenes/agregar-cuenta.webp"
              alt="Agregar Cuenta Web"
              title="Agregar Cuenta Web"
              style={{ width:"60%" }}
            />

            Dale "Ok", cierra esa ventana y ya está configurado.
          </Accordion>
        </AccordionGroup>
      </Step>
      <Step title='En la ventana principal dale en "Añadir instancia"'>
        <img
          src="/tulacraft/imagenes/anadir-instancia.webp"
          alt="Anadir Instancia Web"
          title="Anadir Instancia Web"
          className="mr-auto"
          style={{ width:"21%" }}
        />
      </Step>
      <Step title='Dale a la opción "CurseForge", busca "Tulacraft" y dale "OK"'>
        <img
          src="/tulacraft/imagenes/curseforge-polymc.webp"
          alt="Curseforge Polymc Web"
          title="Curseforge Polymc Web"
          style={{ width:"90%" }}
        />
      </Step>
      <Step title='Ejecuta la instancia y agrega esta IP en "Multiplayer"'>
        ```
        tula.mochos.xyz
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
